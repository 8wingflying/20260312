## 關聯規則挖掘（Association Rule Mining）
- 在大數據庫中找出不同商品或事件之間的「關聯性」。
- 最著名的例子就是「啤酒與尿布」：賣場發現買尿布的爸爸通常也會順便買啤酒，進而調整陳列位置。
- 代表演算法
  - Apriori 演算法
    - 這是最經典且最早被提出的演算法。
    - 它基於一個核心先驗性質（Apriori Property）：「如果一個項目集是頻繁的，那麼它的所有子集也一定是頻繁的。」
    - 反之，如果一個子集不頻繁，其超集也絕不會是頻繁的。
  - FP-Growth 演算法 (Frequent Pattern Growth)為了克服 Apriori 的效率問題，FP-Growth 被開發出來。它最大的特點是不需要產生候選集，且只需要掃描資料庫兩次。
  - Eclat 演算法 (Equivalence Class Transformation)
