# 使用 Python 實例化不同相關係數計算的完整範例
- 主要會用到 pandas（資料處理）和 scipy.stats（統計計算）這兩個函式庫。

## 準備範例資料
- 我們首先建立一個包含各種類型變數的 DataFrame：

```Python
import pandas as pd
import numpy as np
from scipy import stats

# 建立模擬資料
np.random.seed(42)
n = 100

df = pd.DataFrame({
    # 1. 數值型 (線性關係)
    'linear_x': np.linspace(0, 10, n),
    'linear_y': np.linspace(0, 10, n) * 2 + np.random.normal(0, 1, n),
    
    # 2. 數值型 (非線性但單調)
    'nonlinear_y': np.exp(np.linspace(0, 3, n)) + np.random.normal(0, 1, n),
    
    # 3. 次序型 (1-5 分的滿意度)
    'rank_a': np.random.randint(1, 6, n),
    'rank_b': np.random.randint(1, 6, n),
    
    # 4. 名目型與二元型
    'gender': np.random.choice(['Male', 'Female'], n), # 名目
    'is_passed': np.random.choice([0, 1], n),          # 二元 (用於點二系列)
    'score': np.random.normal(70, 10, n)               # 數值 (用於點二系列)
})
```

# 各類相關係數實作

## A. 皮爾森與斯皮爾曼 (數值 vs. 數值)
- 這是最常見的情況，可以直接使用 Pandas 或 Scipy。

```Python
# Pearson (線性)
pearson_corr, _ = stats.pearsonr(df['linear_x'], df['linear_y'])
print(f"Pearson Correlation: {pearson_corr:.3f}")

# Spearman (等級/非線性)
spearman_corr, _ = stats.spearmanr(df['linear_x'], df['nonlinear_y'])
print(f"Spearman Correlation: {spearman_corr:.3f}")
```
## B. 肯德爾 (次序 vs. 次序)
- 適合樣本數較少或有大量重複排名時。

```Python
# Kendall's Tau
kendall_corr, _ = stats.kendalltau(df['rank_a'], df['rank_b'])
print(f"Kendall's Tau: {kendall_corr:.3f}")
```
## C. 點二系列 (二元名目 vs. 數值)
- 例如：性別（男/女）與身高的關係。

```Python
# Point-Biserial
pb_corr, _ = stats.pointbiserialr(df['is_passed'], df['score'])
print(f"Point-Biserial Correlation: {pb_corr:.3f}")
```
## D. Cramer's V (名目 vs. 名目)
- 這需要先建立列聯表（Contingency Table），再計算卡方統計量。

```Python
def calculate_cramers_v(var1, var2):
    # 建立列聯表
    confusion_matrix = pd.crosstab(var1, var2)
    chi2 = stats.chi2_contingency(confusion_matrix)[0]
    n = confusion_matrix.sum().sum()
    phi2 = chi2 / n
    r, k = confusion_matrix.shape
    # 修正偏差後的 Cramer's V
    phi2corr = max(0, phi2 - ((k-1)*(r-1))/(n-1))
    rcorr = r - ((r-1)**2)/(n-1)
    kcorr = k - ((k-1)**2)/(n-1)
    return np.sqrt(phi2corr / min((kcorr-1), (rcorr-1)))

cv_val = calculate_cramers_v(df['gender'], df['is_passed'])
print(f"Cramer's V: {cv_val:.3f}")
```
